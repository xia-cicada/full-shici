# full-shici

全诗词，包含全唐诗、全宋诗和全宋词。

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

> 在打包之前，需要先参考[数据库准备](./docs/build-sql/README.md)来生成poetry.sqlite文件。

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
